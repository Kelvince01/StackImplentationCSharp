# Stack Implentation in C#
## USAGES
### INITIALIZATION
` Stack<T> _stack = new Stack<T>(20); `
> T is a datatype you need to use e.g., int, string, char

### PUSH
` _stack.Push(10); `

### POP
` _stack.Pop(); `
> returns removed element

### PEEK
` _stack.Peek(); `
> returns top element

### CONTAINS
` _stack.Contains(10) `
> returns true if element exists

### COUNT
` _stack.Count(); `
> returns current stack size

### CLEAR
` _stack.Clear(); `
